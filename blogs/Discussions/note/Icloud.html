美区ICloud账户申请

v2talk2022-12-14 08:00:00Discussionsnote

# 前言

由于国内部分法律的缘故，部分APP并不能在国区Store下载和使用，国行iPhone建议日常登录国区Apple ID，备用一个外区Aplle ID 用于登录 App Store.

# 开始

# 准备工作

  1. 能接收验证码的手机号(不限国家)
  2. 尚未注册过Apple ID 的邮箱
  3. iPhone 手机

# 注册美区ID

  1. 访问 美区Apple官网open in new window，滚动页面到右下角是否是 United States，验证官网地址
  2. 点击右上角账户图标，展开选项点击 Account
  3. 点击 Create yours now.open in new window
  4. 生成美国虚拟账单地址，虚拟地址生成器open in new window，选择免税(消费税)州，如 **Alaska(AK) **、特拉华（Delaware），蒙大拿（Montana），新罕布什尔（New Hampshire），俄勒冈（Oregon）
  5. 填写 ‘身份信息’ ，注：出生年月应满足截至注册时大于18周岁
  6. 验证方式，短信验证
  7. 依次验证邮箱和手机号

注：阿拉斯加虽是免税州，但仍有部分城市收取消费税

# 完善账户信息

  1. 登录美区ID账户，管理您的苹果 ID (apple.com)open in new window
  2. 添加付款方式
  3. 国家地区：美国
  4. 付款方式：无
  5. 账单寄送地址信息同上
  6. 勾选 拷贝账单地址
  7. 更新

# 账户激活

  1. 使用iPone进行账号激活
  2. app store 登录账户
  3. 首次登录进行iTunes检查并同意条款

注：iPad并没有SIM身份认证，账单地址付款方式没有None选项，且无法跳过

# 充值

美区商店支付方式仅支持美国信用卡及美区paypal账户，存在一定门槛，推荐采用礼品卡进行充值

注：礼品卡仅支持App Store 商店内进行App的购买，并不支持应用内购

  1. 支付宝首页左上角地区选择 国际/港澳台
  2. 选择 北美洲 / 美国 / 旧金山
  3. 选择 出境
  4. 点击 折扣礼卡
  5. 下拉列表，点击 “更多大牌折扣礼卡”
  6. 搜索 iTunes
  7. 购买时提交的名字可以随意填写，邮箱地址是卡密收件地址

提示：淘宝相关产业链条已经封锁，目前闲鱼尚有余孽，但三方商家可能兜售黑卡，会造成账号封锁
