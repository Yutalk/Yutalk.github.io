# Docker

参考 Docker官方文档open in new window 安装 Docker

RHEL首次安装 Docker Engine-Community之前，需要设置Docker仓库，以便从仓库安装和更新 Docker

设置仓库，需要安装所需的软件包。yum-utils 提供了 yum-config-manager ，并且 device mapper 存储驱动程序需要 device-mapper-persistent-data 和 lvm2。

yum install -y yum-utils device-mapper-persistent-data lvm2

默认没有docker-ce的源，使用yum-config-manager来设置设置Docker仓库yum源为阿里云

yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装docker-ce(社区版)

yum install docker-ce

检查冲突

yum install docker-ce --allowerasing

设置开机自启

systemctl enable docker

安装compose,版本V2.2.2，按需替换版本

$ sudo curl -L "https://github.com/docker/compose/releases/download/v2.2.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

可执行权限

$ sudo chmod +x /usr/local/bin/docker-compose

创建软链：

$ sudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose

测试是否安装成功：

$ docker-compose version

启动容器配置文件

docker-compose up -d
