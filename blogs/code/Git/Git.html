# 什么是 Git

Git（读音为/gɪt/）是一个开源的分布式版本控制系统，高效进行项目版本管理，是 Linus Torvalds 为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件。

Torvalds 开始着手开发 Git 是为了作为一种过渡方案来替代 BitKeeper 。

# GitHub 配置SSH

# 设置邮箱和用户名

打开Git Bash，分别输入下列命令

git config --global user.name "GitHub用户名"

git config --global user.email "GitHub邮箱地址"

# 生成SSH公钥

Git Bash输入下列命令

ssh-keygen -t rsa -C "GitHub邮箱地址"

交互模式，Git Bash 会询问密钥存储路径及密码，一般默认即可

# 将公钥添加到Github

  1. 在~/.ssh/文件夹下找到id_rsa.pub文件，用文本编辑器打开，复制里面的所有内容。
  2. 登陆 github，Add SSH key

# 验证关联成功

Git Bash输入下列命令

ssh -T git@github.com

回复内容如下，即关联成功

Hi [your github username]! You've successfully authenticated

enjoy !
