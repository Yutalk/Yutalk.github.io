# 什么是SSL证书

SSL 证书是一个数字证书，用于认证网站的身份并启用加密连接。SSL 代表安全套接字层，这是一个安全协议，可在 Web 服务器和 Web 浏览器之间创建加密链接。

# Let’s Encrypt

  1. 参考 Let’s Encrypt官网地址open in new window

    我们建议大多数具有命令行访问权限的人使用 Certbotopen in new window ACME 客户端。

  2. 下载certbot

  3. 执行命令(申请通配符证书并不包含根域名，所以添加两条记录)

    certbot certonly --manual --preferred-challenges dns -d example.com -d *.example.com

    说明：

    certonly是子命令，只申请或续约证书，不安装。

    可以让你可以在任意一台联网的PC设备上申请证书，不必是你的服务器

    --manual 选项指以交互或Shell脚本的方式提交信息

    默认是交互方式

  4. 交互模式下，分别配置邮箱、协议许可、是否订阅

  5. 生成两次TXT记录，分别记录到对应域名解析商（不覆盖）

  6. 访问相应地址确认记录生效：toolbox.googleapps.com/apps/dig/#TXT/_acme-challenge.example.com

  7. 记录生效后，Press Enter to Continue

  8. enjoy！

    关于续期

    配置完成后保留TXT记录，再次执行命令完成续期

# ZeroSSL

  1. 网页端申请，ZeroSSLopen in new window

    1. Create SSL Certificate，Click New Certificcate

    2. Enter Domains，www.example.com

    3. Validity，90-day

    4. Auto-Generate CSR

    5. Choose Verification Method：DNS(CNAME)

    6. Fix CNAME record

    7. Verify Domain

    8. Congratulations!

      • 根据Server Type下载证书.zip
      • 配置相关证书
      • 检查安装
      • enjoy!
